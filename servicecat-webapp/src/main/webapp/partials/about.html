Proudly built using the following components (excerpt):
Backend
  • Sparqlify SPARQL-to-SQL rewriter, which includes jena-sparql-api, which includes Apache Jena
  • Java Persistence API backed by Hibernate
Frontend
  • JAvascript Suite for Sparql Access (Jassa)
  • AngularJS and jQuery
Build Environment
  • Apache Maven with the JHipster Plugin which handles NPM, Grunt and Bower
  • Jenkins continuous integration server
Project Hosting
